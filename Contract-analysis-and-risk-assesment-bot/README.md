# Contract Analysis and Risk Assessment Bot

This repository houses the development of a **Contract Analysis & Risk Assessment Bot**. The goal of this project is to assist small and medium businesses in analyzing complex contracts, identifying risks, and automating the generation of legal insights.

## Features
- Contract classification
- Clause identification and risk scoring
- AI-based contextual suggestions and insights
- Clause-to-template matching for compliance

## Installation Steps
1. Clone this repository:
   ```
   git clone https://github.com/Apoorva7705/Contract-analysis-and-risk-assesment-bot.git
   ```
2. Navigate to the **backend** folder and install the dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Navigate to the **frontend** folder:
   ```
   npm install
   ```
4. Run the application:
   - Backend: `uvicorn main:app --reload`
   - Frontend: `npm start`
